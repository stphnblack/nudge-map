About the Plant-Based Nudges Map

Instructions

The map shows places that have enacted plant-based nudges. Click on one of the colored dots for additional information about that place's nudges.

Click the filter or search icons in the top-right to change which places are shown, such as to filter by nudge type. By default, the map shows all places that have adopted any plant-based nudge. You can use the filter to focus on a particular nudge type, such as plant-based defaults. You can also filter the dataset by attributes like institution type or country.

Click the table icon in the top-left to switch from map view to table view. When in table view, click the map icon in the top-left to switch back to the map.
